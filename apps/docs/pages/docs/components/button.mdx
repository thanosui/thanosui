# Button

Buttons allow users to perform actions and choose with a single tap.

```bash
import { Button } from "@thanosui";
```

#### Normal Colors

```ts
type NormalColors =
  | "default"
  | "primary"
  | "secondary"
  | "success"
  | "warning"
  | "error"
  | "gradient";
```
